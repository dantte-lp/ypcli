# Code of Conduct

## Standard

ypcli adopts the Contributor Covenant Code of Conduct, version 2.1, by
reference:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

This policy applies to:

- issues, pull requests, discussions, reviews, and comments in this repository;
- project communication channels linked from this repository;
- public representation of the project by maintainers or appointed
  contributors.

## Enforcement

| Level | Action |
|---|---|
| Clarification | Maintainers request an edit, clarification, or de-escalation. |
| Warning | Maintainers record a warning for behavior that violates this policy. |
| Temporary restriction | Maintainers restrict participation for repeated or severe violations. |
| Permanent restriction | Maintainers block participation for sustained abuse, harassment, or security risk. |

## Reporting

Report conduct violations privately through GitHub:

<https://github.com/dantte-lp/ypcli/security/advisories/new>

Do not include sensitive personal data in public issues or pull requests.
